# 🧠 Agent IA — Mémoire + RAG + Web (local CPU)

Assistant IA **100% local**, tournant sur **CPU**, combinant :
- **TinyLlama** pour la génération de texte (modèle causal léger)
- **MiniLM** pour les embeddings (Sentence-Transformers)
- **ChromaDB** pour la recherche vectorielle (RAG)
- **Streamlit** pour l’interface graphique
- **SQLite** pour la mémoire persistante utilisateur

---

## 🚀 Fonctionnalités principales

- 💬 **Conversation locale** — aucune requête API, tout tourne sur ton CPU
- 💾 **Mémoire persistante SQLite** — profil, humeurs, souvenirs récents
- 📚 **RAG (retrieval-augmented generation)** — ingestion et recherche dans des PDF
- 🌐 **Recherche web optionnelle** — intégration DuckDuckGo
- 🧠 **TinyLlama 1.1B Chat** — modèle compact et rapide pour la génération locale
