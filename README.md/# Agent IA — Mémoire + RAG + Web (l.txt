# Agent IA — Mémoire + RAG + Web (local CPU)

Assistant IA **100% local**, tournant sur **CPU**, combinant :
- **TinyLlama** pour la génération de texte (modèle causal léger)
- **MiniLM** pour les embeddings (Sentence-Transformers)
- **ChromaDB** pour la recherche vectorielle (RAG)
- **Streamlit** pour l’interface graphique
- **SQLite** pour la mémoire persistante utilisateur
